# CapstoneProject
This is a project to use of the NASA's APOD API and save or remove favorites images in local storage

Steps: 
1. Design layout in HTML
2. Add Normalize.css
3. Styling creation
4. Responsive design adjustment
5. API generation
6. Creation of a form that captures the date
7. Programming the API to fetch the image of the day with the form
8. Programming to add and delete favorite images.
